Que es SEO?
SEO

Search Engine Optimization (Optimización para motores de búsqueda). Se trata del conjunto de estrategias y técnicas de optimización que se hacen en una página web para que aparezca orgánicamente en buscadores de Internet como Google, Yahoo o Youtube..



Cómo hacer una web optimizada para SEO?
  1. Darte de alta en Search Console
  2. Hacer un pequeño estudio de palabras clave
  3. Tener los Títulos optimizados
  4. Redactar descripciones con la palabra clave con texto atractivo para convertir


Herramientas para optimizar SEO
  1. Plugins: Yoas SEO o Rank Math
  2. Palabras clave: Ubersuggest, extensión de Chrome Keyword Surfer.
  3. Optimizar imágenes: imagecompressor











Meta Tags

Las etiquetas meta, son fragmentos de código claves que aparecen en el código HTML de una web y que ayudan a los motores de búsqueda a interpretar mejor la página ofreciéndole información extra de la misma.

Nombre Funcionalidad
Título (title) Los buscadores más importantes usan esta etiqueta para titular las entradas en las listas de resultados o SERP, de modo que influye de forma decisiva en la elección del internauta.
Autor (author) y copyright Permiten hacer referencia al diseñador de una página web y al propietario de los derechos del código fuente de una página HTML
Refresh Con este atributo también se define una fecha de vencimiento, a partir de la cual los navegadores o los motores de búsqueda serán dirigidos a una URL establecida previamente.
Etiquetas Hs Como su nombre indica, los encabezados servirán para jerarquizar y ordenar el contenido de una página, por orden de importancia (h1, h2, h3…)
Etiqueta canonical La meta etiqueta canonical es utilizada para decir a Google cual es la página original en el caso de encontrarnos con páginas similares o idénticas.
Etiqueta ALT A la hora de posicionarnos en el navegador, no debemos olvidar el posicionamiento de las imágenes en el navegador. Para esto, haremos uso de la etiqueta ALT (texto alternativo), la cual ayudará al navegador a saber de qué trata nuestra imagen.
Robots meta tag Mediante esta etiqueta, indicaremos al navegador que contenido deber ser indexado y cual no, así como si los enlaces deben ser seguidos o no.
